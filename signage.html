Project

Digital Signage Dashboard

A SaaS CMS for fast-food outlets and advertisers to design layouts, upload content, and launch campaigns.

Overview

This dashboard was developed to eliminate the need for advertisers to call customer service representatives when designing and launching advertisement campaigns. The SaaS platform features an intuitive drag-and-drop system for designing screen layouts, uploading media, and scheduling and launching campaigns — all in one place.

Key Features

  • Analytics page showing campaign activity, media storage summary, and storage space remaining per license.
  • Screen layout designer with drag-and-drop containers for video, images, running text, and more.
  • Media library for uploading and previewing images, videos, and files.
  • Campaign module to tie layouts and media together, then schedule and launch campaigns.
