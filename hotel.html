Project

Hotel Management System

A web platform for hotel receptionists to check-in guests, create room bookings and redeem vouchers.

Overview

This dashboard was developed to digitize the hotel's room booking and voucher system, enabling the hotel manager to easily manage staff and bookings. The analytics module visualizes trends in guest booking activity and preferences, allowing the hotel to make better business decisions based on real data.

Key Features

  • Analytics dashboard showing user activity, number of hotel guests, and membership/voucher redemption summaries.
  • Booking page for creating room bookings with a summary of active, pending, and cancelled bookings.
  • Access control module allowing administrators to restrict user access to relevant features only — e.g. front desk receptionists vs. supervisors.
  • Supervisor controls for adding users, viewing user activity, disabling accounts, and configuring hotel details.
